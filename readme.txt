## How to compile and run
1. Download the zip file and extract to a folder of choice
2. Open your terminal with the path of the AES folder
3. Type in the command "g++ -o aes.cpp aes"
4. Lastly, type in the command "./aes"

## Materials Used
1. Primary source: The FIPS Publication 197
2. Animation of AES on Youtube

## Test Cases
